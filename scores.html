Passages & temps Rox (Hyrox)

Le Start du RUN 1 est l’heure de départ du heat (planning) — non modifiable. Saisis Start et End de chaque segment au format HH:MM ou HH:MM:SS (heure du jour). La colonne Δ = temps de zone Rox : entre la fin du segment (End) et le début du segment suivant (ex. fin de run → début atelier, ou fin d’atelier → début du run suivant). Dernière ligne : pas de segment suivant (—). Coche DNF si l’athlète n’a pas terminé le parcours : la saisie des passages est alors désactivée pour cette ligne (stocké avec les passages, sync cloud). Le bouton Effacer sur chaque ligne remet à zéro uniquement les passages de cet athlète (et le DNF de la ligne).
